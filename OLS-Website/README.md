# Old Laurentian RFC Website

A modern, responsive website for Old Laurentian Rugby Football Club.

## Project Structure

old-laurentian-rfc/
├── index.html              # Main homepage
├── css/
│   ├── style.css          # Main stylesheet
│   └── responsive.css     # Mobile responsiveness
├── js/
│   ├── main.js           # Core JavaScript functionality
│   └── animations.js     # Animation effects
├── images/               # Image assets
│   ├── logo/            # Club logos and branding
│   ├── news/            # News article images
│   ├── players/         # Player photos
│   ├── gallery/         # Photo gallery
│   └── backgrounds/     # Background images
├── pages/               # Additional pages
└── assets/
    └── fonts/           # Custom fonts

## Features
- Responsive design (mobile, tablet, desktop)
- Club branding with official crest
- News and fixtures sections
- Smooth animations and transitions
- Mobile-friendly navigation
- SEO-friendly structure

## Getting Started
1. Install Live Server extension in VS Code
2. Right-click on index.html and select "Open with Live Server"
3. The website will open with live reload

## Customization
### Colors
- --primary-green: #2e5016
- --primary-maroon: #8b1538
- --accent-gold: #f4c430

### Adding Images
1. Place images in appropriate /images/ subfolder
2. Update src attributes in HTML files
3. Optimize images for web
